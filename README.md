# Simple To-Do List

Files:
- index.html — main page
- styles.css — styling
- script.js — app logic (uses localStorage)

How to run:
1. Put all files in the same folder.
2. Open index.html in your browser.

Features:
- Add tasks
- Edit tasks (click text, edit, press Enter or click outside to save)
- Toggle complete
- Delete task
- Filter: All / Active / Completed
- Clear all completed
- Persistent storage via localStorage

Ideas to extend:
- Add due dates and sorting
- Add drag-and-drop to reorder
- Persist to a backend or sync across devices
